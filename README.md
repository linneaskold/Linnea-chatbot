upgift 1
3 saker som fungerar bra

1. Den svarar med ett random response
2. Om den inte hittar något nyckelord så skriver den ett fallbak response.
3. En till bra sak är att den avslutar om man skriver ett farewell word vilket är bra för då kan man avsluta programmet.



3 saker som inte fungerar
1. När jag skriver today så skriver den ut dagens datum fast det är i fel samamanhang.
2. Eftersom att den bara ger ett random svar som den har så kan det bli typ fel svarat, eller fel sammanhang än vad man skriver om.
3. Man kan inte prata om flera saker eftersom den kommer bara välja ett nyckelord om du till exempel har två olika.




uppgift 2

Vad fungerade bra i er chatbot?
Jag tycker att nästan alla responses är ganska bra och den väljer ett random.


Vad fungerade dåligt?
Den kan inte kan svara på sånt som den inte hittar nyckelord till och ger ett fallbak responose vilket gör att det är inte så mycket man kan prata med den egentligen. Sammanhang på saker man skriver med den och svar man får är inte alltid så bra som till exempel om man har två nyckelord och så skriver den ett response men det hänger inte riktigt ihop med vad man skriver.

När förstod chatboten inte användaren?
Om man skriver så att det finns två nyckelord och då passar kanske inte riktigt svaret fullt ut men man kan fortfarande fortsätta konversationen.
